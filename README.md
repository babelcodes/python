# Python

Learn/ing [Python](https://www.python.org)

## Setup on Mac

- Install [PyEnv](https://github.com/pyenv/pyenv) to handle multiple versions of Python:

```
brew update
brew install pyenv

```

- Enable XCode:

```
xcode-select --install
```

- Install wanted version distribution:

```
pyenv install --list
pyenv install 3.7.1
```
